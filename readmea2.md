README – Dice Rolling Game

Game Idea:
This is a simple two-player dice game written in C.
Both players roll a dice in each round, and the number they get is added to their total score.
After all rounds, the player with the higher score wins.

Rules:
• Two players enter their names at the start.
• The user selects how many rounds will be played.
• In each round:
    ▶ Player 1 rolls first.  
    ▶ Player 2 rolls second.
• Each roll generates a random number between 1 and 6.
• The dice face is displayed using ASCII art.
• Scores are updated after every round.
• After all rounds, the player with the highest score wins.

Controls:
• Players only need to enter:
    ▶ Their names  
    ▶ Number of rounds
• After that, the game runs automatically.
• Dice rolls and scores are shown on their own.

Features:
• Dice roll animation using console clearing.   
• ASCII art for dice faces.  
• Live score update every round.  
• Final winner announcement.  
• Uses basic C programming concepts like:
    ▶ Loops  
    ▶ Functions  
    ▶ Random numbers  
    ▶ Console colors
